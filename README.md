# Exercícios de JavaScript (1 a 10)

Este repositório contém um script em Node.js com um menu interativo para executar 10 exercícios básicos de JavaScript, cobrindo:

- Operadores aritméticos
- Operadores de atribuição
- Operadores relacionais
- Operadores lógicos
- Estruturas de decisão (if, else, else if)
- Estruturas de repetição (for, while)
- Vetores (arrays)
- Funções

---

## Pré-requisitos

- **Node.js** instalado (versão 16+ recomendada)

Para verificar:
```bash
node -v
